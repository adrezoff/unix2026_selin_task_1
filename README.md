## Локальный запуск
1. Клонируем репозиторий
```bush
git clone https://github.com/adrezoff/unix2026_selin_task_1.git
```
2. Переходим в корень репозитория
```bush
cd unix2026_selin_task_1
```
3. Компилируем основной алгоритм
```bush
make
```

3. Выдаём права runme на исполнение create_A
```bush
chmod +x runme.sh create_A.sh
```

4. Запускаем скрипт
```bush
./runme.sh
```
